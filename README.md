# House Price Prediction

## Overview
This project aims to predict house prices using machine learning regression techniques.

## Dataset
Boston Housing dataset.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps
- Data Loading
- Data Exploration
- Feature Selection
- Train-Test Split
- Model Training (Linear Regression)
- Model Evaluation

## Evaluation Metrics
- Mean Squared Error (MSE)
- R2 Score

## Visualization
A scatter plot was used to compare actual vs predicted house prices.

## Results
The model was able to learn patterns in the data and provide reasonable predictions.
